# VRMモデルの配置について

このディレクトリ (`public/models/`) に対象のVRMモデルを配置してください。

**対象モデル**: `ふらすこ式風きりたん_VRM_1_0_1.vrm`
**配置先ファイル名**: `kiritan.vrm` (ファイル名を変更して配置してください)

## ライセンスに関する注意点
本プロジェクトでは「ふらすこ式風東北きりたん」を使用想定としていますが、**モデルの再配布は禁止**されています。
そのため、GitHubや配布パッケージにモデル本体を同梱することはできません。
ユーザー自身で規約に同意の上、モデルを入手し、本ディレクトリに手動で配置してください。
