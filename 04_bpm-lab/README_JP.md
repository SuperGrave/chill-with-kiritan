# BPM Comparison Lab

キリタン壁紙の現行BPM検出と、別方式・合議方式を同じ音源で比較する独立サンプルです。壁紙本体やCompanionの設定・データは変更しません。

## 起動

リポジトリ直下の `Run_BPM_Lab.bat` をダブルクリックします。初回だけ依存パッケージを自動インストールし、`http://127.0.0.1:5190` を開きます。

## 入力

- 音声ファイル: ファイル選択またはドラッグ＆ドロップ。ブラウザ内で再生します。
- PC音声: 画面共有ダイアログで共有対象を選び、「音声を共有」をONにします。
- マイク: Windowsで選択されている入力を解析します。
- 合成テスト: 88 / 120 / 174 BPM、および低音のない120 BPMを生成します。

## 比較方式

1. `LOW-BAND IOI`: 壁紙v0.8.8と同じ、低域4バケットのオンセット＋拍間隔中央値。
2. `SPECTRAL FLUX`: 全帯域の正方向変化からオンセットを検出し、同じ拍間隔推定へ入力。
3. `AUTOCORRELATION`: 約12秒のオンセット強度履歴から繰り返し周期を推定。
4. `2/3 CONSENSUS`: 0.5倍・2倍テンポを正規化し、3方式中2方式以上の一致を採用。

曲名と既知の正解BPMを入力して「現在値を記録」を押すと、方式ごとの誤差を表へ保存できます。CSV出力はブラウザのダウンロード先へ保存されます。

## 注意

- ブラウザのPC音声キャプチャ可否は共有対象とDRMに依存します。比較を再現しやすいのは音声ファイル入力です。
- 約12秒窓方式と5秒ロックがあるため、曲の安定した区間を15秒以上流してから記録してください。
- このラボの結果を確認してから、採用する方式だけを壁紙本体へ移植します。
