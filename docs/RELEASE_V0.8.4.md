# v0.8.4 Release Notes

v0.8.4はCompanionの小型起動、日常操作の短縮、個人ニュース再生、配布初期値をまとめた更新。

## Changes

- Companionを最小許容サイズの960×600で起動。
- ホームの「いま更新」から、歌詞なし時の個人ニュース自動表示を切替可能。
- 歌詞なしへの切替時は個人ニュースを中断位置から再生し、歌詞復帰時はその位置で一時停止。
- 個人ニュースのリピートをデフォルトONに変更。
- 左ナビの「操作・中身・調律・環境・更新」ラベルを非表示化。
- 表示設定の6分類を「3Dモデル・背景・パネル表示・視点・モーション・システム」に改名。
- 初回プリセットを1920×1080にし、1920×1200と1920×1080の保存済みサンプルを同梱。
- 個人ニュースの初回サンプルを『銀河鉄道の夜』と2026年7月10日ニュースに整理。
- 表示プリセット上書き前に確認ダイアログを追加。

公開パッケージは従来どおりVRM/VRMAを含まない。

## 公開ファイルの区分

- そのまま使う方向け: 全部入りzip、Wallpaper Engine用zip、Companionインストーラー／ポータブル版
- 改造・開発する方向け: `Chill-with-Kiritan-v0.8.4-source.zip`

制作版zipはコンパイル済み成果物、VRM/VRMA、APIキー、認証情報、個人設定・原稿を含まない。セットアップとビルド方法は同梱の `README_SOURCE_JP.md` を参照。
