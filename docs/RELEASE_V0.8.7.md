# v0.8.7 Release Notes

v0.8.7は、v0.8.6公開後の自動起動と個人用ニュース表示の改善に加え、卓上アンビエントモーション5本を追加したリリースです。

## 主な変更

- 自動起動の状態確認、タスク登録、Run keyフォールバックをコンソール非表示で実行。
- 設定保存後に自動起動登録を重複実行していた経路を解消。
- UAC確認は「管理者権限で登録」を明示的に選んだ場合だけ表示。
- 個人用ニュースの通常行を一つの長いBLOCKへ連結し、電光掲示板のように一列で最後まで表示。
- `[Break]`、`[Paragraph]`、`[段落]`、`---`、`[Topic: ...]`、`(Wait: ...)` を明示的な表示境界として使用可能。
- 各BLOCKのマーキーは一度だけ流れ、画面外まで完走してから次のBLOCKへ移行。
- 旧LINE形式の再生位置を、絶対経過時間から新BLOCK位置へ復元。
- 個人用ニュースのマーキーをrAF駆動の等速ピクセルスクロールへ書き換え。CSSアニメーションの状態同期による250msごとの再シーク（カクつき）と、途中打ち切り／早期完走の原因だった表示側スクロール速度スケーリング・180秒クランプを撤廃。BLOCKの全文が右端から入り左端へ完全に抜けるまでを、BLOCKの持ち時間ちょうど（約5.6文字/秒の読みペース）で走破する。
- ※これに伴い、表示設定の「横スクロール速度」は原稿の読みペース（文字数比例＋`(Wait: ...)`指定）が速度を決めるため表示には影響しなくなった。

## 追加アンビエントモーション（卓上・小物なし5本）

- `amb_work_wrist_flex` 手首をほぐす（通常作業）: タイピングを止めて両手を胸元に上げ、手首を反らして左右にほぐし、キーへ戻る。
- `amb_work_window_gaze` ふと遠くを見る（通常作業）: 手はキーに置いたまま、目線の高さの遠くをぼーっと眺めて画面へ戻る。
- `amb_work_window_gaze_mirror` ふと逆側を見る（通常作業）: 上記の鏡写し。反対の奥側を向く。
- `amb_vid_drowse` うとうと→ハッと目を覚ます（動画視聴）: 頬杖のまま頭が沈み、ハッと起きて照れ笑い。深夜帯に出やすい。
- `amb_slp_mumble` 寝言をつぶやく（睡眠）: 腕枕のまま口がもごもご動き、ふにゃっと安らぐ。目は閉じたまま。

全モーションはライブランタイムで境界シームレス（手首ジャンプ2mm前後以下）、ハンドピン維持（頬杖の顎↔掌46mm±2mm等）を実測検証済み。

## 個人用ニュースの書式例

```text
[Topic: 今日の話]
この行と
この行は一つの長い文章としてつながって流れます。

[Break]
ここから次の段落として流れます。
```

空行だけでは表示を分割しません。次へ切り替えたい場所に区切り記号を書いてください。
