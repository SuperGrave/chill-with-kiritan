# v0.8.9 Release Notes

v0.8.9は、v0.8.8で追加したSPECTRUMパネルとBPM同期イベント境界の上に、きりたん本体のBPM連動モーション「音楽ノリノリモード」と、確定BPMへのユーザー補正（±10）を実装したリリースです。

## 音楽ノリノリモード（music_listen）

- 固定モード「音楽ノリノリ」を追加（Companionの「きりたんの動き方 → 固定 → 音楽ノリノリ」。自動ローテーションには入らないため、音楽が鳴っていない時に勝手にこのモードへは入りません）。
- 待機モーション `loop_music_listen`: 右腕はタイピング姿勢の静止版（手首はキーボード接地IKで固定）、左手は新しい手形状 `ear_cup` で左耳に手のひらをかざして聴き入ります。左腕はライブランタイムの9パラメータFKソルバで校正し、手のひらは耳側（頭側面向き誤差3.4°）、肘は真横〜やや後方（z-0.06）に置いて袖がカメラ視線に掛からない形にしています。
- リズムモーションはクリップ再生ではなくビート位相からの毎フレーム計算のため、BPMが変わると次のフレームから速度が追従します。
  - 横揺れ: 112BPM以下限定・2拍で1往復。腰から上だけが揺れ、接地した右手首はIKが吸収。テンポが上がるほど振幅を絞ります。
  - 指トントン: 毎拍（150BPM超は2拍に1回）、手のひらと指を一緒に持ち上げて拍の頭で落とす打楽器型。
  - 2つのフィギュアは8小節ごとにクロスフェードで交代し、グルーヴ中は微笑み、BPMロスト時は待機ポーズへ戻ります。

## BPM補正（±10）

- 確定（ロック）BPMにユーザー好みの補正値（-10〜+10、整数）を加算できます。検知は常に生信号で行い、補正はパネル表示と `kiritan:audio-bpm-sync` の `bpm` にだけ反映されます。
- 補正中のパネル表示は `BPM 123 (+3)` のように内訳を併記。
- ロック中に補正値を変えると、再検知なしで同じロックのまま新しいテンポを即時再送信します。
- `kiritan:audio-bpm-sync.detail` に `rawBpm`（補正前）と `bpmOffset` を追加（`bpm` は補正後の最終テンポ）。
- 設定はOverlayの SPECTRUM → RHYTHM SYNC「BPM Offset (±10)」、Companionの「スペクトラム内容 → BPM補正（±10）」。

## BPMラボ（04_bpm-lab）

- 低域ビート間隔・スペクトルフラックス・自己相関・コンセンサスの4方式を同じ音源で比較する独立ラボを追加（ルートの `Run_BPM_Lab.bat` で起動）。本体ビルドには含まれません。
