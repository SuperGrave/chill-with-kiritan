# v0.9.0 Release Notes

v0.9.0は、BPM推定ときりたんの音楽連動を実用段階へ進め、個人ニュースを電光掲示板のような一続きの表示へ作り直したリリースです。

## BPM推定とスペクトラム設定

- 低域ビート間隔・スペクトルフラックス・自己相関の3方式と、それらをまとめるコンセンサス方式を搭載。
- 同じテンポが3〜12秒続いた時点でBPMを確定。使用する判定方式、確定待ち時間、BPM補正（±10）をCompanionから選択可能。
- Overlayから各方式の推定値・確信度・合議数を毎秒Companionへ送り、STUDIOの「スペクトラム」でリアルタイムに比較可能。
- BPMが確定していない時や音声入力が止まった時は、古い値を同期中として扱わず「検知中／停止中」へ戻ります。

## 音楽ノリノリモード

- `music_listen`を自動ローテーションの低頻度候補へ追加。不要な場合は「通常モードで出さないモード」から除外できます。
- 左手を耳の横へかざす待機ポーズを再校正し、手の甲を外側、袖を体の後方へ配置。
- BPM確定中は横揺れまたは指トントンをビート位相へ同期。32拍ごとにフィギュアを切り替えます。
- 首振りを拍全体に広がる滑らかなコサイン運動へ変更。150 BPM超では2拍周期に落として細かな振動を防ぎます。
- モーション確認パネルから40〜240 BPMを任意注入でき、実音源なしでも同期状態を確認できます。

## 個人ニュース連続ティッカー

- 文ごとに画面外から出し直す方式を廃止し、文章ブロックを`◆`でつないだ一列のコンベア表示へ変更。
- 短文だけが急加速する問題、文末後の空白、250ms更新境界での停止や位置飛びを解消。
- 明示的な段落・トピック・待機指定でのみ次ブロックへ進み、通常の文章は最後まで一続きに表示します。
- 末尾から先頭へ戻るループ継ぎ目にも仮想コピーを配置し、巻き戻りや一瞬の停止なしで循環します。

## 同梱プリセット

- 1920×1200用レイアウトを現在の実機調整値へ更新し、スペクトラムパネルを右カラムへ配置。
- 1920×1080用サンプルも同じ構成を画面高に合わせて調整。
