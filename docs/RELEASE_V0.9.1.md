# v0.9.1 Release Notes

v0.9.1は、検出BPMの短い揺れや途切れで、きりたんのBPM連動モーションが停止・再始動を繰り返す問題を改善する安定化リリースです。

## 5 BPM固定モーションバンク

- 40〜240 BPMを5 BPM刻みで用意した固定速度のモーションバンクへ変更。
- 検出値をフレームごとにそのまま速度へ反映せず、最も近い5 BPM帯（例: 128 BPMは130 BPM）を選びます。
- 同じ帯域内でBPMが再推定されても、モーションの位相・フィギュア・再生位置をリセットしません。
- 別の帯域へ移る場合も拍内の位相を保ち、横揺れと指トントンの切り替えはクロスフェードします。
- 個々のビート検出による位相の引っ張りを廃止し、ノイズや細かな打音で身体が跳ねる現象を防ぎます。

## BPMロスト猶予

- BPMロックが外れても、選択中のモーションを設定秒数だけ継続します。
- 猶予中に同じ5 BPM帯へ再ロックした場合は、停止や最初からの再生をせず、そのまま継続します。
- 猶予時間はCompanionの「スペクトラム」から0〜30秒で調整可能。初期値は8秒です。
- 0秒へ設定すると、従来と同じくロック喪失時に即停止します。

## 検証

- BPM推定・モーション結合テストで、固定バンク間隔、丸め、同一帯域再ロック、別帯域への位相維持、ビートノイズ耐性、猶予満了を確認。
- Wallpaper、Overlay、CompanionのTypeScriptビルドとRustテストを実施。
